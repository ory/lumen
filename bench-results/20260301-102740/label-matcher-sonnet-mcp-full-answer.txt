## Usage Example

```go
// Exact equality: job="prometheus"
m, err := NewMatcher(MatchEqual, "job", "prometheus")

// Regex: env=~"prod.*"
m, err := NewMatcher(MatchRegexp, "env", "prod.*")
```

The key design detail: `FastRegexMatcher` is only initialized for the two regexp types; equality matchers have `re == nil` and methods like `Prefix()`, `SetMatches()`, and `IsRegexOptimized()` return zero values for them.
