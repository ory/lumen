## `MatchType` — the four matcher kinds

```go
type MatchType int

const (
    MatchEqual     MatchType = iota  // "="
    MatchNotEqual                    // "!="
    MatchRegexp                      // "=~"
    MatchNotRegexp                   // "!~"
)
```

---

## `Matcher` struct

```go
type Matcher struct {
    Type  MatchType
    Name  string
    Value string

    re *FastRegexMatcher  // populated only for regexp types
}
```

---

## Constructors

**`NewMatcher`** — primary constructor, returns an error if regexp compilation fails:

```go
func NewMatcher(t MatchType, n, v string) (*Matcher, error)
```

- For `MatchRegexp` / `MatchNotRegexp`: compiles the value via `NewFastRegexMatcher` and stores it in `re`.
- For `MatchEqual` / `MatchNotEqual`: `re` is `nil`.

**`MustNewMatcher`** — panics on error, for use in tests only:

```go
func MustNewMatcher(mt MatchType, name, val string) *Matcher
```
