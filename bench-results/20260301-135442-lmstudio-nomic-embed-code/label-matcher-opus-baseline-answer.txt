### Constructor — `NewMatcher` (line 56-70)

```go
func NewMatcher(t MatchType, n, v string) (*Matcher, error) {
    m := &Matcher{Type: t, Name: n, Value: v}
    if t == MatchRegexp || t == MatchNotRegexp {
        re, err := NewFastRegexMatcher(v)
        if err != nil {
            return nil, err
        }
        m.re = re
    }
    return m, nil
}
```

It sets the basic fields, then **only compiles a `FastRegexMatcher`** when the type is `MatchRegexp` or `MatchNotRegexp`. For equality types, `re` stays nil.

There's also a convenience wrapper `MustNewMatcher` (line 73) that panics on error, intended for tests.
