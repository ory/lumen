### Constructor

```go
// labels_matcher.go:55-70
func NewMatcher(t MatchType, n, v string) (*Matcher, error) {
    m := &Matcher{
        Type:  t,
        Name:  n,
        Value: v,
    }
    if t == MatchRegexp || t == MatchNotRegexp {
        re, err := NewFastRegexMatcher(v)
        if err != nil {
            return nil, err
        }
        m.re = re
    }
    return m, nil
}
```

Key behavior: for `MatchRegexp` and `MatchNotRegexp`, `NewMatcher` compiles the value string into a `FastRegexMatcher` at construction time and stores it in the unexported `re` field. For equality types (`MatchEqual`/`MatchNotEqual`), `re` is left nil. There's also `MustNewMatcher` (`labels_matcher.go:72-79`) which panics on error, intended only for tests.
