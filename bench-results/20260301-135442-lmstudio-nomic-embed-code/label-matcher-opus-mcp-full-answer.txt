### MatchType enum

```go
type MatchType int

const (
    MatchEqual     MatchType = iota  // "="
    MatchNotEqual                     // "!="
    MatchRegexp                       // "=~"
    MatchNotRegexp                    // "!~"
)
```

Four match types using `iota`: exact equality, not-equal, regex match, and negated regex match. Each has a string representation (`=`, `!=`, `=~`, `!~`) via the `matchTypeToStr` array and the `String()` method.

Creates a `Matcher` with the given type, label name, and value. If the type is `MatchRegexp` or `MatchNotRegexp`, it compiles a `FastRegexMatcher` from the value string (returning an error if the regex is invalid). For equality types, no regex is compiled.
